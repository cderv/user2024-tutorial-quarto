A penguin story

Author

Christophe Dervieux

Published

July 8, 2024

Data

For this analysis, we’ll use the penguin dataset in the R package palmerpenguins (https://allisonhorst.github.io/palmerpenguins).

library(tidyverse)
library(palmerpenguins)
library(gt)

The penguins dataset contains the following elements:

glimpse(penguins)

Species

We have 3 species in this dataset: Adelie, Gentoo, and Chinstrap that are not evenly distributed.

penguins |>
  count(species) |>
  knitr::kable()
species n
Adelie 152
Chinstrap 68
Gentoo 124

The figure below shows the differences in culmen by species.

ggplot(
  penguins,
  aes(
    x = bill_length_mm, y = bill_depth_mm,
    color = species, shape = species
  )
) +
  geom_point() +
  labs(x = "Culmen length (mm)", y = "Culmen depth (mm)")
